# DHiDiB Foundation - Youth Empowerment Platform

A modern, responsive website for the DHiDiB Foundation, a youth-led nonprofit organization dedicated to empowering young people with skills, knowledge, and opportunities for a sustainable future.

## 🌟 Features

### 🎨 Modern Design & UI
- **Glass-morphism Design**: Sophisticated blur effects and transparency
- **Responsive Layout**: Optimized for all devices (mobile, tablet, desktop)
- **Smooth Animations**: CSS animations and transitions throughout
- **Custom Typography**: Mercy Christole font for elegant headings
- **Gradient Effects**: Beautiful color transitions and visual appeal

### 📱 Enhanced Mobile Experience
- **Advanced Hamburger Menu**: Solid black blurry background with maximum readability
- **Touch-Friendly Interface**: Optimized for mobile interactions
- **Smooth Transitions**: Fluid animations for menu interactions
- **Professional Navigation**: Clean, modern mobile navigation design

### 🚀 Interactive Components
- **Hero Section**: Animated background with floating particles
- **Feature Cards**: Hover effects and gradient backgrounds
- **Event Calendar**: Upcoming events with registration functionality
- **Blog Section**: Interactive blog posts with modal views
- **Contact Forms**: Volunteer and donation forms with validation

### 🎯 Core Sections
- **Home**: Hero section with call-to-action buttons
- **Features**: How to make impact with interactive cards
- **About Us**: Mission, vision, and core values
- **Get Involved**: Volunteer and donation forms
- **Programs**: Youth empowerment initiatives
- **Events**: Upcoming and past events
- **Blog**: Latest articles and insights
- **Contact**: Contact information and social links

## 🛠️ Technology Stack

- **Framework**: Next.js 15.4.6 with App Router
- **Styling**: Tailwind CSS with custom animations
- **Language**: TypeScript for type safety
- **Fonts**: Mercy Christole (custom) + system fonts
- **Icons**: SVG icons and emojis
- **Forms**: GetForm.io integration for form handling
- **Deployment**: Vercel-ready configuration

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm, yarn, pnpm, or bun

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/abdifatah-com/Dhidib-Site.git
cd Dhidib-Site
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. **Run the development server**
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000) to see the result.

### Running on Custom Port
```bash
npx next dev -p 3001
```
